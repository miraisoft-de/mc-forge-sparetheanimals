Spare The Animals version 2.2-mc1.21.3 by miraisoft

License: CC BY miraisoft

This is a data pack for Minecraft 1.21.3!



You like animals? You don't wanna kill them? Even in a game? This data pack helps you avoid killing innocent animals in Minecraft.
Books are essential, so you need leather. You can barely obtain leather without killing animals in Minecraft Vanilla.

This data pack makes zombies drop leather, skeletons drop feathers.
There are also new recipes for black dye, ink sacs and glowing ink sacs.



Installation:

Extract this zip file into the datapacks subfolder of your world.
You can also choose data packs during world creation.

Also read the tutorial about data packs in the Minecraft wiki if you need more information.
